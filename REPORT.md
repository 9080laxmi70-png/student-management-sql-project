## Objective
To build a Student Management Database and analyze student performance using SQL queries.

## Database Design
The database contains a Students table storing student details such as age, gender, grade, and subject scores.

## Query Explanation

1. Display all student details  
   Shows all records from the Students table.

2. Average score per subject  
   Calculates average marks for Math, Science, and English.

3. Top performer  
   Identifies the student with the highest total score.

4. Student count per grade  
   Groups students by grade and counts them.

5. Average score by gender  
   Compares performance between male and female students.

6. Students with Math score above 80  
   Filters students who performed well in Mathematics.

7. Update student grade  
   Updates student information using the UPDATE statement.

## Conclusion
This project demonstrates fundamental SQL skills including database creation, data insertion, aggregation, and filtering.
